BanglaDOC Surya Clean - Quick Commands (macOS + Windows + Docker)
===================================================================

⚡ FOR WINDOWS USERS: See WINDOWS-SPECIFIC section below
===============================================

===============================================
WINDOWS 10/11 - QUICKSTART (NATIVE SETUP)
===============================================

0) Prerequisites - Install Once
-------------------------------
- Python 3.12+ (https://www.python.org/downloads/)
- Git Bash or PowerShell
- PostgreSQL 15+ OR Docker
- VS Code

1) Clone and navigate (PowerShell)
----------------------------------
git clone <repo>
cd bangladoc_surya_clean

2) First-time setup (PowerShell)
-------------------------------
# Copy environment
copy backend\.env.example backend\.env

# Create virtual environment
python -m venv backend/venv
backend/venv/Scripts/Activate.ps1

# Install dependencies
python -m pip install --upgrade pip
python -m pip install -e "./backend[dev]"

# Start PostgreSQL (Services or Docker)
# If Docker: docker run -d --name bangladoc-db -e POSTGRES_USER=bangladoc -e POSTGRES_PASSWORD=arka -e POSTGRES_DB=bangladoc -p 5432:5432 postgres:15

3) Daily startup (PowerShell - 3 terminals)
-------------------------------------------
# Terminal 1: API Server
cd bangladoc_surya_clean/backend
backend/venv/Scripts/Activate.ps1
uvicorn bangladoc_ocr.server.app:app --reload --host 0.0.0.0 --port 8000

# Terminal 2: Celery Worker
cd bangladoc_surya_clean/backend
backend/venv/Scripts/Activate.ps1
python -m celery -A bangladoc_ocr.celery_app:celery_app worker --loglevel=info --pool=solo -n worker1@%h

# Terminal 3: Ollama (optional but recommended)
ollama serve

# Then open browser: http://localhost:8000

4) Test it works
---------------
# PowerShell: Upload PDF via browser at http://localhost:8000
# Or CLI:
cd bangladoc_surya_clean/backend
backend/venv/Scripts/Activate.ps1
bangladoc C:\path\to\test.pdf --verbose

5) Stop everything
------------------
# Ctrl+C in each terminal
# Then: docker stop bangladoc-db (if using Docker for DB)

===============================================
WINDOWS - DOCKER SETUP (Even Easier!)
===============================================

0) Install Docker Desktop (https://www.docker.com/products/docker-desktop)

1) Start everything (PowerShell)
-------------------------------
cd bangladoc_surya_clean
docker-compose build
docker-compose up -d

2) Check services
-----------------
docker ps
curl http://localhost:8000/health

3) Open UI
---------
start http://localhost:8000

4) Stop everything
------------------
docker-compose down
