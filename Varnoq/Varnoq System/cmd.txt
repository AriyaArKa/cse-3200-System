========================================
macOS - Step by Step
========================================

1) First-time setup (one time)
------------------------------
cd "/Users/arkabrajaprasadnath/3-2/System PRoject/cse-3200-System/Varnoq/Varnoq System"
cp backend/.env.example backend/.env
# Use only this project env path: backend/venv (do not create another .venv elsewhere)
python3 -m venv backend/venv
source backend/venv/bin/activate
python -m pip install --upgrade pip setuptools wheel
python -m pip install -e "./backend[dev]"

# Optional for Apple Silicon
export PYTORCH_ENABLE_MPS_FALLBACK=1

2) Start PostgreSQL locally
---------------------------
brew services start postgresql

3) Start Redis locally
----------------------
brew services start redis

4) Start Ollama (optional)
--------------------------
ollama serve

5) Start API (new terminal)
---------------------------
cd "/Users/arkabrajaprasadnath/3-2/System PRoject/cse-3200-System/Varnoq/Varnoq System/backend"
source venv/bin/activate
uvicorn bangladoc_ocr.server.app:app --reload --host 0.0.0.0 --port 8000

6) Start worker (new terminal)
------------------------------
cd "/Users/arkabrajaprasadnath/3-2/System PRoject/cse-3200-System/Varnoq/Varnoq System/backend"
source venv/bin/activate
python -m celery -A bangladoc_ocr.celery_app:celery_app worker --loglevel=info --pool=solo -n worker1@%h

7) Open app
-----------
open http://127.0.0.1:8000

8) Quick checks
---------------
curl -s http://127.0.0.1:8000/health
curl -s http://127.0.0.1:8000/stats

9) Stop
-------
# Stop API/worker/ollama with Ctrl+C
brew services stop redis
brew services stop postgresql
